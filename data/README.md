# Митний кодекс

`customs_code.db` — локальна SQLite-база з офіційного HTML-файлу Верховної Ради України.

У базі є:

- `customs_sections` — розділи;
- `customs_chapters` — глави;
- `customs_articles` — статті та службові записи для пунктів Розділу XXI;
- `customs_articles_fts` — пошуковий індекс SQLite FTS5;
- `document_meta` — службова інформація про джерело й редакцію.

Поточне джерело: `customs_code_source.htm`, редакція 2026-01-04.

Поточна структура бази:

- 21 розділ;
- 85 глав;
- 746 записів у `customs_articles`: 671 офіційна стаття + 75 пунктів Розділу XXI `ПРИКІНЦЕВІ ТА ПЕРЕХІДНІ ПОЛОЖЕННЯ`.

Розділ XXI не має офіційних глав і статей у тексті кодексу, тому для єдиної структури застосунку він імпортується як службова глава `XXI`, а його пункти зберігаються як записи `XXI-1`, `XXI-2`, `XXI-9-1` тощо.

Для повторного створення бази:

```bash
python scripts/import_customs_code_from_html.py
python scripts/check_customs_code.py
```
